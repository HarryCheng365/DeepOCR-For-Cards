我们的 产品特征

移动PC两不误，快速判定精准识别

移动适配

Html5 和 Angular JS结合 用户可自由选择拍照还是从手机图库上传。利用JQuery的JCrop插件，实现对图片的裁剪。裁剪完成后生成新图通过Angular JS的 ng-file-upload 插件上传

数据存储

Spring boot后台从HttpServletRequest中取出来图片，并通过JPA接口将图片路径存储在数据库中

消息递送

选择通过socket建立Java客户端和Python服务端之间的通信，当用户点击上传图片时，将其转换为byte序列，向服务端发起链接请求，并传送数据。服务端接收完数据后，利用send方法将识别信息或错误信息进行返回，由客户端本地将结果转换为json。接收数据时通过settimeout()方法解决socket调用recv接收数据时因为无法处理tcp数据流结束而阻塞的问题。

图片切割

图像切割部分两条基于openCV的道路殊途同归，第一条路是Gaussian去噪后二值化处理，用Sobel算子对X、Y方向进行边缘检测，随后根据测试经验利用膨胀算法使得校园卡的横线得以更好连接，随后根据轮廓进行切割；第二条路使用直接阈值法二值化的方法对深色背景照片进行快速操作。随后我们利用approxPolyDP( )进行长方形拟合，用getPerspectiveTransform( )投影消除畸变，根据卡片信息区域、头像区域、樱花城堡区域相对位置进行切割并根据结果判定是否为校园卡。

模型识别

OCR识别三头并进，一条路为Ctpn+Crnn实现文字定位和文字识别，一条路为谷歌的PyTesseract库，第三条路为成熟的百度api。同时对识别结果进行了语义模型匹配，对关键字、字符串长度等特定条件进行限制，并使用Try-except获取最优的结果，随后将结果从阿里云服务器Server端返回Client端。
